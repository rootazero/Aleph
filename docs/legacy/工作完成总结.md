## 🎯 技术总结

### 崩溃根本原因
```
线程 6 (rdev listener):
  rdev::macos::listen::raw_callback
  → rdev::macos::keyboard::Keyboard::string_from_code
  → TSMGetInputSourceProperty (macOS 输入法 API)
  → _dispatch_assert_queue_fail ❌ 崩溃
```

**问题**：rdev 0.5.x 在后台线程调用 `TSMGetInputSourceProperty`
**要求**：macOS 要求此 API 必须在主线程调用
**修复**：rdev 0.6.0 正确处理主线程要求

### 代码变更
```diff
# Aleph/core/Cargo.toml
- rdev = "0.5"
+ rdev = { git = "https://github.com/Narsil/rdev.git", branch = "main" }
```
