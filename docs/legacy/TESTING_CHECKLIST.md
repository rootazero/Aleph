## 在 Xcode 中运行应用

### 1. 编译和运行

1. **选择目标**:
   - 在 Xcode 顶部工具栏，选择 `Aether` scheme
   - 选择 `My Mac` 作为运行目标

2. **编译并运行**:
   - 按 `Cmd + R` 或点击 ▶️ 按钮
   - 等待编译完成（首次编译可能需要几分钟）

3. **检查启动**:
   - 应用不会显示在 Dock（设置了 LSUIElement=YES）
   - 应在菜单栏右上角看到 ✨ 图标

### 2. 核心功能测试

#### 2.1 菜单栏功能

- [ ] **菜单栏图标**: 点击菜单栏 ✨ 图标，检查下拉菜单
  - [ ] 显示 "Settings..." 菜单项
  - [ ] 显示 "About" 菜单项
  - [ ] 显示 "Quit" 菜单项

#### 2.2 设置界面

点击 "Settings..." 打开设置窗口：

**窗口设计**:
- [ ] **交通灯按钮**: 自定义红黄绿三色按钮在左侧圆角侧边栏顶部
  - [ ] 红色按钮悬停显示 ✕ 图标，点击关闭窗口
  - [ ] 黄色按钮悬停显示 − 图标，点击最小化窗口
  - [ ] 绿色按钮悬停显示 ⤢ 图标，点击切换全屏
- [ ] **圆角侧边栏**: 左侧导航栏有 18pt 圆角
- [ ] **无原生标题栏**: 内容紧贴窗口上沿
- [ ] **窗口拖拽**: 可以从内容区域拖动窗口

**侧边栏导航**:
- [ ] General 标签
- [ ] Providers 标签
- [ ] Routing 标签
- [ ] Shortcuts 标签
- [ ] Behavior 标签
- [ ] Memory 标签（如果已实现）

#### 2.3 Providers 管理

切换到 "Providers" 标签：

**Provider 列表**:
- [ ] **显示预设 Providers**: OpenAI, Claude, Gemini, Ollama
- [ ] **Provider 卡片**: 显示名称、类型、状态
- [ ] **启用/禁用开关**: 可以切换 provider 的启用状态
- [ ] **默认 Provider 标记**: 显示哪个是默认 provider

**添加新 Provider**:
- [ ] 点击 "Add Provider" 按钮
- [ ] 弹出编辑面板
- [ ] **Provider 类型选择**: 可选择 OpenAI, Claude, Ollama, Custom
- [ ] **表单字段**:
  - [ ] Provider Name (必填)
  - [ ] API Key (OpenAI/Claude 必填)
  - [ ] Model Name
  - [ ] Base URL (可选)
  - [ ] Color Picker (选择 provider 颜色)
  - [ ] Timeout (秒)
  - [ ] Max Tokens
  - [ ] Temperature (0.0-2.0)
- [ ] **测试连接**: 点击 "Test Connection" 按钮
  - [ ] 显示加载状态
  - [ ] 成功显示 ✓ 图标和提示
  - [ ] 失败显示错误信息
- [ ] **保存**: 点击 "Save" 保存配置

**编辑 Provider**:
- [ ] 点击 provider 卡片的 "Edit" 按钮
- [ ] 修改配置
- [ ] 保存更改

**删除 Provider**:
- [ ] 点击 provider 卡片的 "Delete" 按钮
- [ ] 确认删除对话框
- [ ] Provider 从列表中移除

#### 2.4 Routing 规则

切换到 "Routing" 标签：

**规则列表**:
- [ ] **显示路由规则**: 每条规则显示正则表达式和目标 provider
- [ ] **拖动排序**: 可以拖动规则改变优先级
- [ ] **启用/禁用**: 可以切换规则的启用状态

**添加规则**:
- [ ] 点击 "Add Rule" 按钮
- [ ] **表单字段**:
  - [ ] Pattern (正则表达式)
  - [ ] Target Provider (下拉选择)
  - [ ] System Prompt (可选)
- [ ] **验证**: 测试正则表达式是否有效
- [ ] **保存**: 添加到规则列表

**编辑/删除规则**:
- [ ] 点击规则的 "Edit" 按钮编辑
- [ ] 点击规则的 "Delete" 按钮删除

#### 2.5 Shortcuts 配置

切换到 "Shortcuts" 标签：

- [ ] **全局快捷键**: 显示当前快捷键（默认 Cmd+~）
- [ ] **快捷键录制**: 点击输入框，按下新的快捷键组合
- [ ] **冲突检测**: 检测与系统快捷键的冲突
- [ ] **重置**: 恢复默认快捷键

#### 2.6 Behavior 设置

切换到 "Behavior" 标签：

- [ ] **输入模式**:
  - [ ] Cut (剪切模式)
  - [ ] Copy (复制模式)
- [ ] **输出模式**:
  - [ ] Instant (立即粘贴)
  - [ ] Typewriter (打字机效果)
- [ ] **打字速度**: 调整滑块（仅 Typewriter 模式）

#### 2.7 配置持久化

**测试配置保存**:

**验证 API Key 存储**:
- [ ] 添加一个 provider 并输入 API key
- [ ] 打开 macOS "钥匙串访问" 应用
- [ ] 搜索 "aether"
- [ ] **验证**: API key 存储在 Keychain 中
- [ ] **验证**: config.toml 中没有明文 API key
