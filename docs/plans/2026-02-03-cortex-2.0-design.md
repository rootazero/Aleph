# Aleph Cortex 2.0 Design

> **Status**: Approved
> **Created**: 2026-02-03
> **Authors**: Architecture Review

---

## Executive Summary

Aleph Cortex 2.0 is a comprehensive refactoring of the dispatcher module's internal implementation, addressing five critical issues identified in the architecture review:

| Priority | Issue | Root Cause | Solution |
|----------|-------|------------|----------|
| **P0** | JSON parsing fragility | Greedy regex matching | StreamParser (state machine) |
| **P0** | Prompt injection vulnerability | Unsanitized user input | SecurityPipe (sanitizer rules) |
| **P1** | Scattered configuration | Hardcoded thresholds | DecisionEngine (unified config) |
| **P1** | Missing CN PII protection | US-only regex patterns | SecurityPipe (locale-aware masking) |
| **P2** | Token budget overflow | No cost accounting | TokenEconomist (tiktoken + RAG) |

**Deployment Strategy**: Strangler Fig Pattern - preserve external API, replace internals incrementally.

```rust
// core/src/dispatcher/cortex/security/sanitizer.rs

/// Unified trait for sanitization rules
pub trait SanitizerRule: Send + Sync {
    /// Rule name (for logging and config)
    fn name(&self) -> &str;

    /// Rule priority (lower number executes first)
    fn priority(&self) -> u32;

    /// Execute sanitization, return processed text + triggered flag
    fn sanitize(&self, input: &str, ctx: &SanitizeContext) -> SanitizeResult;

    /// Whether enabled (can be dynamically disabled via config)
    fn is_enabled(&self, config: &SecurityConfig) -> bool { true }
}

/// Sanitization context (passes session info)
pub struct SanitizeContext {
    pub session_id: Option<String>,
    pub locale: Locale,           // zh-CN, en-US, etc.
    pub user_trust_level: TrustLevel, // Optional: trusted users skip some checks
}

/// Sanitization result
pub struct SanitizeResult {
    pub output: String,           // Processed text
    pub triggered: bool,          // Whether rule triggered
    pub action: SanitizeAction,   // Action taken
    pub details: Option<String>,  // Trigger details (for audit logs)
}

pub enum SanitizeAction {
    Pass,                         // Pass without modification
    Masked(usize),                // Masked N sensitive occurrences
    Escaped(usize),               // Escaped N dangerous markers
    ConfidencePenalty(f32),       // Lower confidence (rather than reject)
    Blocked(String),              // Block directly, with reason
}
```

```rust
pub struct SecurityPipeline {
    rules: Vec<Box<dyn SanitizerRule>>,
    config: Arc<RwLock<SecurityConfig>>,
    watcher: Option<FileWatcher>,  // YAML rule file watcher
}

impl SecurityPipeline {
    /// Execute complete sanitization pipeline
    pub fn process(&self, input: &str, ctx: &SanitizeContext) -> PipelineResult {
        let mut current = input.to_string();
        let mut actions = Vec::new();

        // Execute in priority order
        for rule in self.rules.iter().filter(|r| r.is_enabled(&self.config.read())) {
            let result = rule.sanitize(&current, ctx);
            if result.triggered {
                actions.push((rule.name().to_string(), result.action.clone()));
                current = result.output;

                // If blocked, return immediately
                if matches!(result.action, SanitizeAction::Blocked(_)) {
                    return PipelineResult::blocked(result);
                }
            }
        }

        PipelineResult::ok(current, actions)
    }
```

#### 3.3 Hot-Reload & Config Manager

```rust
// core/src/dispatcher/cortex/decision/mod.rs

use notify::{Watcher, RecursiveMode};
use std::sync::Arc;
use tokio::sync::{watch, RwLock};

pub struct DecisionEngine {
    /// Global config (hot-reload)
    global_config: Arc<RwLock<DecisionConfig>>,
    /// Config change broadcast (notify all sessions)
    config_tx: watch::Sender<DecisionConfig>,
    /// Session override table
    session_overrides: Arc<RwLock<HashMap<String, SessionOverride>>>,
}

impl DecisionEngine {
    /// Start config file watching
    pub fn start_watching(&self, config_path: PathBuf) -> Result<(), ConfigError> {
        let global = self.global_config.clone();
        let tx = self.config_tx.clone();

        let mut watcher = notify::recommended_watcher(move |res| {
            if let Ok(event) = res {
                if event.kind.is_modify() {
                    if let Ok(new_config) = load_config(&config_path) {
                        *global.blocking_write() = new_config.clone();
                        let _ = tx.send(new_config);
                        tracing::info!("DecisionConfig hot-reloaded");
                    }
                }
            }
        })?;

        watcher.watch(&config_path, RecursiveMode::NonRecursive)?;
        Ok(())
    }

    /// Get effective config for session
    pub async fn get_effective_config(&self, session_id: &str) -> DecisionConfig {
        let global = self.global_config.read().await;
        let overrides = self.session_overrides.read().await;

        match overrides.get(session_id) {
            Some(override_) => merge_config(&global, override_),
            None => global.clone(),
        }
    }

    /// Subscribe to config changes (for session real-time sync)
    pub fn subscribe(&self) -> watch::Receiver<DecisionConfig> {
        self.config_tx.subscribe()
    }
}
```

```rust
use std::sync::Arc;

/// Token counter trait (supports multi-provider adaptation)
pub trait TokenCounter: Send + Sync {
    /// Count tokens in text
    fn count(&self, text: &str) -> usize;

    /// Count tokens in message list (including role marker overhead)
    fn count_messages(&self, messages: &[Message]) -> usize;

    /// Get encoder name
    fn encoder_name(&self) -> &str;
}

/// Default counter based on tiktoken (cl100k_base)
pub struct TiktokenCounter {
    encoder: tiktoken_rs::CoreBPE,
}

impl TiktokenCounter {
    pub fn cl100k() -> Self {
        Self {
            encoder: tiktoken_rs::cl100k_base().unwrap(),
        }
    }

    pub fn for_model(model: &str) -> Self {
        // Claude, GPT-4, etc. all use cl100k compatible encoding
        // Future: extend to model -> encoder mapping
        Self::cl100k()
    }
}

impl TokenCounter for TiktokenCounter {
    fn count(&self, text: &str) -> usize {
        self.encoder.encode_with_special_tokens(text).len()
    }

    fn count_messages(&self, messages: &[Message]) -> usize {
        // Each message has fixed overhead (role markers, etc.)
        const MESSAGE_OVERHEAD: usize = 4; // <|role|> tokens

        messages.iter()
            .map(|m| self.count(&m.content) + MESSAGE_OVERHEAD)
            .sum()
    }

    fn encoder_name(&self) -> &str {
        "cl100k_base"
    }
}

/// Counter registry (select by provider)
pub struct CounterRegistry {
    counters: HashMap<String, Arc<dyn TokenCounter>>,
    default: Arc<dyn TokenCounter>,
}

impl CounterRegistry {
    pub fn get(&self, provider: &str) -> Arc<dyn TokenCounter> {
        self.counters.get(provider)
            .cloned()
            .unwrap_or_else(|| self.default.clone())
    }
}
```

#### 4.2 Budget Manager

```rust
// core/src/dispatcher/cortex/budget/mod.rs

/// Token budget config
#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct BudgetConfig {
    /// Soft threshold for tool list injection
    pub tool_injection_threshold: usize,  // default: 4000
    /// Tool count threshold triggering RAG fallback
    pub rag_tool_count_threshold: usize,  // default: 20
    /// Number of tools to retrieve in RAG mode
    pub rag_top_k: usize,                 // default: 8
    /// Tokens reserved for LLM response
    pub response_reserve: usize,          // default: 4096
}

/// Token budget manager
pub struct TokenEconomist {
    counter: Arc<dyn TokenCounter>,
    config: BudgetConfig,
    tool_selector: Arc<dyn ToolSelector>,
}

/// Budget calculation result
#[derive(Debug)]
pub struct BudgetReport {
    /// System prompt token count
    pub system_tokens: usize,
    /// History messages token count
    pub history_tokens: usize,
    /// Tool definitions token count
    pub tool_tokens: usize,
    /// Total
    pub total: usize,
    /// Model context window
    pub context_limit: usize,
    /// Remaining available
    pub available: usize,
    /// Recommended injection strategy
    pub strategy: InjectionStrategy,
}

#[derive(Debug, Clone, PartialEq)]
pub enum InjectionStrategy {
    /// Full injection of all tools
    FullInjection,
    /// RAG mode: only inject relevant tools
    RagMode { selected_tools: Vec<String> },
    /// Over limit: need to truncate history
    TruncateHistory { keep_last: usize },
    /// Severely over: reject execution
    OverBudget { overflow: usize },
}

impl TokenEconomist {
    /// Calculate budget and return recommended strategy
    pub async fn calculate_budget(
        &self,
        system_prompt: &str,
        messages: &[Message],
        available_tools: &[UnifiedTool],
        context_limit: usize,
    ) -> BudgetReport {
        let system_tokens = self.counter.count(system_prompt);
        let history_tokens = self.counter.count_messages(messages);

        // Calculate tool definition token overhead
        let tool_tokens: usize = available_tools.iter()
            .map(|t| self.estimate_tool_tokens(t))
            .sum();

        let total = system_tokens + history_tokens + tool_tokens;
        let available = context_limit.saturating_sub(total + self.config.response_reserve);

        // Decide injection strategy
        let strategy = self.decide_strategy(
            tool_tokens,
            available_tools.len(),
            available,
            messages,
        ).await;

        BudgetReport {
            system_tokens,
            history_tokens,
            tool_tokens,
            total,
            context_limit,
            available,
            strategy,
        }
    }

    async fn decide_strategy(
        &self,
        tool_tokens: usize,
        tool_count: usize,
        available: usize,
        messages: &[Message],
    ) -> InjectionStrategy {
        // Case 1: Tool count or tokens exceed threshold → RAG mode
        if tool_count > self.config.rag_tool_count_threshold
            || tool_tokens > self.config.tool_injection_threshold
        {
            let query = Self::extract_query_from_messages(messages);
            let selected = self.tool_selector
                .select_relevant(&query, self.config.rag_top_k)
                .await;
            return InjectionStrategy::RagMode {
                selected_tools: selected,
            };
        }

        // Case 2: Insufficient available space → truncate history
        if available < self.config.response_reserve {
            let keep = Self::calculate_keep_count(messages, available);
            return InjectionStrategy::TruncateHistory { keep_last: keep };
        }

        // Case 3: Normal → full injection
        InjectionStrategy::FullInjection
    }

    fn estimate_tool_tokens(&self, tool: &UnifiedTool) -> usize {
        // Estimate: name + description + schema
        let schema_str = serde_json::to_string(&tool.input_schema)
            .unwrap_or_default();
        self.counter.count(&tool.name)
            + self.counter.count(&tool.description)
            + self.counter.count(&schema_str)
            + 20 // Formatting overhead
    }
}
```

```rust
// core/src/dispatcher/cortex/budget/tool_selector.rs

/// Tool selector trait (reserved interface for RAG fallback)
#[async_trait]
pub trait ToolSelector: Send + Sync {
    /// Select most relevant top-k tools based on query
    async fn select_relevant(&self, query: &str, top_k: usize) -> Vec<String>;
}

/// Vector similarity-based tool selector
pub struct VectorToolSelector {
    embedder: Arc<dyn Embedder>,
    tool_embeddings: RwLock<HashMap<String, Vec<f32>>>,
}

impl VectorToolSelector {
    /// Pre-compute embeddings for all tools
    pub async fn index_tools(&self, tools: &[UnifiedTool]) -> Result<(), EmbedError> {
        let mut embeddings = self.tool_embeddings.write().await;
        for tool in tools {
            let text = format!("{}: {}", tool.name, tool.description);
            let embedding = self.embedder.embed(&text).await?;
            embeddings.insert(tool.name.clone(), embedding);
        }
        Ok(())
    }
}

#[async_trait]
impl ToolSelector for VectorToolSelector {
    async fn select_relevant(&self, query: &str, top_k: usize) -> Vec<String> {
        let query_embedding = self.embedder.embed(query).await.ok();

        let embeddings = self.tool_embeddings.read().await;
        let mut scores: Vec<_> = embeddings.iter()
            .filter_map(|(name, emb)| {
                query_embedding.as_ref().map(|q| {
                    let score = cosine_similarity(q, emb);
                    (name.clone(), score)
                })
            })
            .collect();

        scores.sort_by(|a, b| b.1.partial_cmp(&a.1).unwrap());
        scores.into_iter().take(top_k).map(|(name, _)| name).collect()
    }
}

/// Simple keyword matching selector (no embedding needed, as fallback)
pub struct KeywordToolSelector {
    tool_keywords: HashMap<String, Vec<String>>,
}

#[async_trait]
impl ToolSelector for KeywordToolSelector {
    async fn select_relevant(&self, query: &str, top_k: usize) -> Vec<String> {
        let query_lower = query.to_lowercase();
        let mut scores: Vec<_> = self.tool_keywords.iter()
            .map(|(name, keywords)| {
                let score = keywords.iter()
                    .filter(|kw| query_lower.contains(*kw))
                    .count();
                (name.clone(), score)
            })
            .filter(|(_, score)| *score > 0)
            .collect();

        scores.sort_by(|a, b| b.1.cmp(&a.1));
        scores.into_iter().take(top_k).map(|(name, _)| name).collect()
    }
}
```

```rust
// core/src/dispatcher/cortex/error.rs

use thiserror::Error;

/// Cortex unified error type
#[derive(Debug, Error)]
pub enum CortexError {
    // === Parser errors ===
    #[error("JSON parse failed: {message}")]
    ParseError {
        message: String,
        raw_input: String,
        recovery_attempted: bool,
    },

    #[error("Schema validation failed for tool '{tool}': {reason}")]
    SchemaValidationError {
        tool: String,
        reason: String,
        expected_schema: Option<String>,
    },

    // === Security errors ===
    #[error("Input blocked by security rule '{rule}': {reason}")]
    SecurityBlocked {
        rule: String,
        reason: String,
        severity: SecuritySeverity,
    },

    #[error("PII detected and masked: {count} occurrences")]
    PiiMasked { count: usize },  // Warning, not fatal

    // === Decision errors ===
    #[error("No tool matched (confidence: {confidence:.2})")]
    NoMatch { confidence: f32 },

    #[error("Confirmation timeout after {timeout_ms}ms")]
    ConfirmationTimeout { timeout_ms: u64 },

    #[error("User rejected tool execution")]
    UserRejected,

    // === Budget errors ===
    #[error("Context overflow: {overflow} tokens over limit")]
    ContextOverflow { overflow: usize },

    #[error("Token counting failed: {reason}")]
    TokenCountError { reason: String },

    // === Config errors ===
    #[error("Config load failed: {path}")]
    ConfigLoadError {
        path: String,
        #[source]
        source: std::io::Error,
    },

    #[error("Config validation failed: {reason}")]
    ConfigValidationError { reason: String },
}

#[derive(Debug, Clone, Copy, PartialEq)]
pub enum SecuritySeverity {
    Low,      // Possible false positive
    Medium,   // Suspected attack
    High,     // Confirmed attack
    Critical, // Must block
}

/// Recovery hint (for upper layer decision)
impl CortexError {
    pub fn recovery_hint(&self) -> RecoveryHint {
        match self {
            Self::ParseError { recovery_attempted: false, .. } => {
                RecoveryHint::RetryWithRepair
            },
            Self::NoMatch { confidence } if *confidence > 0.2 => {
                RecoveryHint::FallbackToChat
            },
            Self::ContextOverflow { .. } => {
                RecoveryHint::TruncateAndRetry
            },
            Self::ConfirmationTimeout { .. } => {
                RecoveryHint::PromptUserAgain
            },
            _ => RecoveryHint::Abort,
        }
    }
}

#[derive(Debug, Clone)]
pub enum RecoveryHint {
    RetryWithRepair,    // Attempt repair and retry
    FallbackToChat,     // Fallback to general chat
    TruncateAndRetry,   // Truncate history and retry
    PromptUserAgain,    // Re-prompt user
    Abort,              // Cannot recover, terminate
}
```
